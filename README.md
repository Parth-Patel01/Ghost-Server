## 🌟 About SoulStream

SoulStream is a mystical media streaming platform where your videos become eternal souls flowing through the digital realm. Experience the ultimate dark-themed streaming service with advanced features like pause/resume uploads, background processing, and soul-like UI animations.

### ✨ Key Features

- 🌊 **Soul Portal Upload**: Creative dark-themed upload interface with mystical animations
- ⏸️ **Pause/Resume Powers**: Full control over your upload souls - pause and resume anytime
- 📱 **Background Soul Flow**: Uploads continue even when you minimize the browser
- 🎨 **Eternal Dark Theme**: Beautiful black background with red/purple accents
- 🔄 **Session Restoration**: Automatically restore upload progress on page reload
- 📊 **Soul Progress Tracking**: Real-time animated progress bars with soul-flow effects
- 🎬 **Streaming Sanctuary**: Full-featured video player with HLS support
- 💾 **Persistent Soul Storage**: Background uploads saved to localStorage
- 🌐 **Mobile Soul Experience**: Fully responsive design for all devices

### 🎯 Perfect For

- 🏠 **Home Media Servers**: Turn your Raspberry Pi into a mystical streaming portal
- 🎥 **Content Creators**: Manage your video collection with style
- 🌙 **Dark Theme Enthusiasts**: Experience the beauty of darkness
- 🔮 **Unique UI Lovers**: Enjoy creative animations and mystical branding

## 🚀 Quick Start

### 1. Clone the Soul Repository
```bash
git clone https://github.com/yourusername/soulstream.git
cd soulstream
```

### 2. Install Soul Dependencies
```bash
npm install
cd frontend && npm install
```

### 3. Start the Soul Flow
```bash
npm start
```

### 4. Access the Portal
Open your browser and navigate to `http://localhost:3000`

## 🔧 Advanced Installation (Raspberry Pi)

For the full SoulStream experience on Raspberry Pi:

```bash
chmod +x scripts/install.sh
./scripts/install.sh
```

This will set up:
- 🌊 Complete SoulStream service
- 🔧 Nginx reverse proxy
- 📱 Mobile-optimized experience
- 🎬 HLS streaming support
- 💾 SQLite database
- 🔄 Auto-restart capabilities

## 🛠️ Technical Architecture

### Backend Soul Engine
- **Node.js**: Powers the soul server
- **Express.js**: Handles soul routing  
- **SQLite**: Stores soul metadata
- **Multer**: Manages soul uploads
- **FFmpeg**: Processes video souls

### Frontend Soul Interface
- **React**: Creates the soul UI
- **Tailwind CSS**: Styles the darkness
- **Heroicons**: Mystical icons
- **HLS.js**: Streams the souls

### Soul Features
- **Chunked Uploads**: Break large souls into manageable pieces
- **Rate Limiting**: Separate limits for general (100/15min) and upload (10,000/15min) APIs
- **Retry Logic**: Exponential backoff for failed soul transfers
- **Session Management**: Persistent upload sessions
- **Background Processing**: Souls continue flowing even when tab is hidden
